# FreelanceFlow Frontend

AI-powered business management platform for freelancers — Frontend Application built with Next.js 15.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4, shadcn/ui
- **State & HTTP**: TanStack Query, Axios
- **Form Management**: React Hook Form, Zod
- **Animations & Styling**: Framer Motion, Lucide Icons
- **Charts**: Recharts

## Setup Instructions

1. **Environment Variables**:
   Copy `.env.example` to `.env.local` and configure:
   ```bash
   cp .env.example .env.local
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
   Loads the frontend locally at `http://localhost:3000`.

4. **Build Production**:
   ```bash
   npm run build
   ```
